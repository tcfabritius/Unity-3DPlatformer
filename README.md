# 🎮 Unity 3D Platformer

A 3D platformer built in Unity by following YouTube tutorials — a self-teaching project made before starting formal game development studies at Metropolia University of Applied Sciences.

---

## About

This project was created independently as an introduction to Unity and C# game development, guided by online tutorials. It served as a first hands-on exploration of 3D game mechanics, character movement, and Unity's rendering pipeline before any formal coursework began.

It is not a polished or complete game — it reflects an early learning phase and is kept here as part of a development journey.

---

## What I Learned

- Unity Editor basics and scene setup
- 3D character controller and movement
- C# scripting in Unity
- Working with Unity's component system
- Version control for Unity projects with Git

## Getting Started

### Prerequisites

- [Unity](https://unity.com/download) — check `ProjectSettings/ProjectVersion.txt` for the exact version
- Git

### Running the Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/tcfabritius/Unity-3DPlatformer.git
   ```

2. **Open in Unity**
   - Launch Unity Hub
   - Click **Open** → select the cloned project folder
   - Unity will import assets and resolve packages automatically

3. **Play**
   - Open the main scene from the `Assets/` folder
   - Press the **Play** button in the Unity Editor
